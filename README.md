# Getting Started

1. Make sure ffmpeg is installed locally on machine
2. `npm install` or `yarn`
3. Configure .env
4. Run: `node index.js`

### ENV Example

```
LOCAL_FILE_URI=""
LOCAL_FILE_URI2=""
```

# Overview

- Compress video
- Transcode video
- Split video to M3U8 Playlist for video on demand / streaming playback
- Extract MP3 from MP4
- Make GIF preview from MP4
